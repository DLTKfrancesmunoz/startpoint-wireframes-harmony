## Adding New Patterns

### Automated (Recommended)

```bash
# From existing component
python scripts/create_pattern.py \
  --name "My Pattern" \
  --product "PPM" \
  --category "dialogs" \
  --component-path "src/components/MyComponent.tsx"

# Interactive mode
python scripts/create_pattern.py --interactive

# From wizard component (shortcut)
python scripts/create_pattern.py --from-wizard "src/components/wizards/MyWizard.tsx"
```

### Manual

1. Copy `reference/PATTERN_TEMPLATE.md`
2. Fill in all sections
3. Save to `patterns/{product}/{category}/{pattern-name}.md`
4. Add entry to this registry

## Pattern Status Workflow

```
draft → review → approved
                    ↓
              deprecated
```

- **draft**: Initial documentation, may be incomplete
- **review**: Ready for team review
- **approved**: Accepted for use in production
- **deprecated**: Being phased out, don't use for new work

## Searching Patterns

```bash
# List all patterns
python scripts/search_patterns.py --list

# Filter by product
python scripts/search_patterns.py --product "PPM"

# Filter by category
python scripts/search_patterns.py --category "navigation"

# Keyword search
python scripts/search_patterns.py --query "wizard accordion"

# Full-text search
python scripts/search_patterns.py --query "sidebar completion" --full-text

# Summary statistics
python scripts/search_patterns.py --summary
```
